## Getting Started  

### Prerequisites  
- Docker or Podman ([Install Guide](https://docs.docker.com/get-docker/))

### Installation  

1. Clone the repository:

```bash
git clone https://github.com/xhandlr/note-forge.git
```

2. Start the stack:

```bash
cd note-forge && podman-compose up
```
